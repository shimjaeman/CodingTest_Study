## 4. 규칙 📜
### One Day 
* 코딩 테스트 1문제 이상 문제풀이하고 저녁 12시전까지 인증할 것

### Weekend 
* 매주 정해진 공부범위를 공부 후 정리해서 일요일 12시전까지 인증할 것
* 주 1회 전주 학습 상태 점검(노트 필기 or 개인 sns 페이지 등) 및 리뷰 시간

### Penalty (미정)
* one day : 1000원
* weekend : 3000원
